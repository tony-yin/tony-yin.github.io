tech
  • Ceph
  • RGW

本文通过ceph-deploy安装和创建RGW，然后分别创建S3和Swift接口并提供了相应的方案。

Ceph RGW基于librados，是为应用提供RESTful类型的对象存储接口。RGW提供两种类型的接口：

  • S3：兼容Amazon S3 RESTful API
  • Swift：兼容OpenStack Swift API

S3和Swift API共享同一个命名空间，所以可以使用两种API访问相同的数据。

小结
